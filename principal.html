¿Que es github?

la plataforma Github es un programa que se usa comúnmente para integrar código y almacenar archivos para que puedan almacenarse en carpetas. A menudo se utiliza para crear programas de computadora. La versión actual de Github se basa en Fireworks Pillars. Una empresa fundada en 2010, el último lugar es git hub, que se creó originalmente para el uso de sistemas electrónicos.
